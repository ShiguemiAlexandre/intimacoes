# Instalação do Poetry

```bash
    pip install poetry
```

# Instalando dependências
```bash
    poetry install
```

# Adicionando dependências
```bash
    poetry add nome-da-dependência
```

# Ativando o ambiente virtual
```bash
    poetry shell
```

# Desativando ambiente virtual

```bash
    exit
```

# Atualizando Dependências

```bash
    poetry update
```
